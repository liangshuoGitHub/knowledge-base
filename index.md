## 我怎么使用它

你可以把这个仓库理解成两层：
- [`knowledge-base-repo`](./README.md) 是内容源
- [`VitePress`](https://vitepress.dev/) 是浏览器里的展示层

本地查看时，优先通过文档站访问，而不是直接在文件树里一篇篇点开。

## 快速开始

```bash
npm install
npm run docs:dev
```

启动后，在浏览器中打开本地地址即可查看最新内容。
